J’ai réalisé une application web full stack appelée AI Resume Analyzer avec React. Elle permet à l’utilisateur d’uploader son CV en PDF, de saisir une offre d’emploi, puis l’application analyse le CV avec une intelligence artificielle pour donner un score ATS et des conseils d’amélioration. Techniquement, j’ai utilisé React, TypeScript et Vite pour le frontend, et Puter.js comme backend cloud pour gérer l’authentification, le stockage des fichiers et l’appel à l’IA. Quand l’utilisateur envoie son CV, je le stocke dans le cloud, je le convertis en image, puis je l’envoie à l’API IA avec un prompt structuré. La réponse est enregistrée et affichée sous forme de scores et de recommandations claires dans l’interface
